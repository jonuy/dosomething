Detailed Description

Defines a batch queue.

Stale items from failed batches are cleaned from the {queue} table on cron using the 'created' date.

Member Function Documentation

BatchQueue::claimItem ( $  lease_time = 0  ) 

Overrides SystemQueue::claimItem().

Unlike SystemQueue::claimItem(), this method provides a default lease time of 0 (no expiration) instead of 30. This allows the item to be claimed repeatedly until it is deleted.

Reimplemented from SystemQueue.
