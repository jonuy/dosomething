\subsection{Detailed Description}
Displays a list of forum topics.

Available variables:
\begin{DoxyItemize}
\item \$header: The table header. This is pre-\/generated with click-\/sorting information. If you need to change this, see \hyperlink{forum_8module_a1030c21aad210d17887edaea6a5aeb28}{template\_\-preprocess\_\-forum\_\-topic\_\-list()}.
\item \$pager: The pager to display beneath the table.
\item topics}
An array of topics to be displayed. Each topic in \$topics contains:
\begin{DoxyItemize}
\item \$topic-\/$>$icon: The icon to display.
\item \$topic-\/$>$moved: A flag to indicate whether the topic has been moved to another forum.
\item \$topic-\/$>$title: The title of the topic. Safe to output.
\item \$topic-\/$>$message: If the topic has been moved, this contains an explanation and a link.
\item \$topic-\/$>$zebra: 'even' or 'odd' string used for row class.
\item \$topic-\/$>$comment\_\-count: The number of replies on this topic.
\item \$topic-\/$>$new\_\-replies: A flag to indicate whether there are unread comments.
\item \$topic-\/$>$new\_\-url: If there are unread replies, this is a link to them.
\item \$topic-\/$>$new\_\-text: Text containing the translated, properly pluralized count.
\item \$topic-\/$>$created: A string representing when the topic was posted. Safe to output.
\item \$topic-\/$>$last\_\-reply: An outputtable string representing when the topic was last replied to.
\item \$topic-\/$>$timestamp: The raw timestamp this topic was posted.
\end{DoxyItemize}
\item \$topic\_\-id: Numeric ID for the current forum topic.
\end{DoxyItemize}
