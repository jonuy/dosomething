_filter_url ( $  text,
$  filter 
)

Implements callback_filter_process().

Converts text into hyperlinks automatically.

This filter identifies and makes clickable three types of "links".

  • URLs like http://example.com.
  • E-mail addresses like name@example.com.
  • Web addresses without the "http://" protocol defined, like www.example.com. Each type must be processed separately, as there is no one regular expression that could possibly match all of the cases in one pass.
_filter_url_escape_comments ( $  match,
$  escape = NULL 
)

Escapes the contents of HTML comments.

Callback for preg_replace_callback() within _filter_url().

Parameters:
$match An array containing matches to replace from preg_replace_callback(), whereas $match[1] is expected to contain the content to be filtered.
$escape (optional) A Boolean indicating whether to escape (TRUE) or unescape comments (FALSE). Defaults to NULL, indicating neither. If TRUE, statically cached $comments are reset.
