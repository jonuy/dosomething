ctools_plugin_example_default_page_manager_pages (  ) 

Default panels pages for CTools Plugin Example

To pick up this file, your module needs to implement hook_ctools_plugin_api() - See ctools_plugin_example_ctools_plugin_api() in ctools_plugin_example.module.

Note the naming of the file: <modulename>.pages_default.inc With this naming, no additional code needs to be provided. CTools will just find the file. The name of the hook is <modulename>_default_page_manager_pages()

This example provides two pages, but the returned array could have several pages.

Returns:
Array of pages, normally exported from Panels.
