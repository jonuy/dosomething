Detailed Description

ctools_access_ruleset module

This module allows styles to be created and managed on behalf of modules that implement styles.

The ctools_access_ruleset tool allows recolorable styles to be created via a miniature scripting language. Panels utilizes this to allow administrators to add styles directly to any panel display.
