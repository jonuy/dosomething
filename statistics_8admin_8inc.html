statistics_access_log ( $  aid  ) 

Page callback: Gathers page access statistics suitable for rendering.

Parameters:
$aid The unique accesslog ID.
Returns:
A render array containing page access statistics. If information for the page was not found, drupal_not_found() is called.
statistics_recent_hits (  ) 

Page callback: Displays the "recent hits" page.

This displays the pages with recent hits in a given time interval that haven't been flushed yet. The flush interval is set on the statistics settings form, but is dependent on cron running.

Returns:
A render array containing information about the most recent hits.
statistics_top_pages (  ) 

Page callback: Displays statistics for the "top pages" (most accesses).

This displays the pages with the most hits (the "top pages") within a given time period that haven't been flushed yet. The flush interval is set on the statistics settings form, but is dependent on cron running.

Returns:
A render array containing information about the the top pages.
